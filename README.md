# Hung Pham — Data Portfolio

A sleek, dark-themed Next.js portfolio for **Pham Quoc Hung (GTRe5)** — Data Enthusiast.

## 🎨 Design
- **Theme**: Data Noir — deep dark background, electric cyan accents, terminal aesthetics
- **Fonts**: Space Mono (monospace) + Syne (display) + Inter (body)
- **Animations**: Typewriter effect, animated skill bars, counter stats, scan-line overlay

## 📋 Sections
1. **Hero** — Animated typewriter roles, stats counters, CTA
2. **About** — Personal bio, traits, avatar placeholder
3. **Skills** — Animated progress bars by category + tools cloud
4. **Projects** — 6 data project cards with impact metrics
5. **Experience** — Timeline with 3 roles + education sidebar
6. **Contact** — Working contact form + social links

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🛠 Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Google Fonts** (Space Mono, Syne, Inter)

## 🔧 Customization
- Update personal info in each component
- Replace avatar placeholder in `components/About.tsx`
- Add real project links in `components/Projects.tsx`
- Connect contact form to a backend (e.g., Resend, Formspree)
- Update social links in `components/Contact.tsx`
